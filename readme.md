This project is part of the IBM Data Science Professional Certificate capstone.

The objective is to analyze SpaceX launch data and build machine learning models to predict whether the first stage of a Falcon 9 rocket will successfully land, enabling cost estimation and decision support.

## Project Structure

- `data/`
  - `raw/`: Original datasets
  - `processed/`: Cleaned and transformed data
- `notebooks/`: Data collection, EDA, and machine learning notebooks
- `visuals/`: Plots and visual outputs
- `docs/`: Project documentation and final report
- `app/`: Interactive dashboard application

## Key Topics
- Data Collection (API & Web Scraping)
- Exploratory Data Analysis
- Feature Engineering
- Machine Learning Classification
- Data Visualization & Dashboards
